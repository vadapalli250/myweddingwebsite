My Little Experiments

Its rightly quoted in witty vow's story that Marriage is like a roller coaster ride and It teaches Life in hard way.

I am no different. I did sign up for my marriage lessons but determined to start the learning a bit different. To learn a different subject first. So i have taken up this little project to design my own wedding website and host it in cloud infrastructure.

When i got this idea, i was pretty excited and started designing my webpage. Well, it turned out to be beautiful and elegant on paper. Then comes the implementation so do realization. I have started learning front end technologies for the first time and quickly took a deep dive in 5 different programming languages. Soon realized web developers out there didnt't get the status overnight and then started looking for a alternatives. After multiple failures, I have got this beautiful template to start my dream learning.

During this journey i have started learning 3 programming languages, refreshed my knowledge on 10+ cloud services, got to know many ways of doing single job and important thing is, i took my first step towards my dream being a front end developer.
